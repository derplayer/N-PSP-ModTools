# N-PSP-ModTools
N+ PSP ModTools - Old project from 2014

Converts the "raw" 16-bit ABGR 4444 Texture into editable DDS format.

Drag and drop a "raw" texture from the game into executeable. You'll get an DDS that now can be edited.
When youre finnished with editing drag'n drop it back to convert the DDS to "raw";
Swap the file in the game image out and you're finnished.

For more look into repository wiki.
